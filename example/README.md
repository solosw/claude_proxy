## 项目列表

### 1. 俄罗斯方块 (tetris.html)

经典俄罗斯方块游戏的纯原生实现！

**特点：**
- 原生 JavaScript + HTML5 Canvas 实现，无需任何依赖
- 现代渐变配色设计
- 完整的游戏功能：
  - 方块预览（Next Piece 显示）
  - 实时分数统计
  - 游戏等级提升
  - 行消除动画效果
- 响应式布局设计

**操作说明：**
| 按键 | 功能 |
|------|------|
| ← → | 左右移动方块 |
| ↓ | 加速下落 |
| ↑ | 旋转方块 |
| 空格 | 快速落地 |
| P | 暂停/继续 |
| R | 重新开始 |
